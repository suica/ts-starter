# ts-starter

Template for TypeScript Project

## Features

- TypeScript
- Jest
- Lint-staged via husky, with Eslint & Prettier
- ESM
- VSCode config (auto format on save, auto organize imports)

## Commands

- `pnpm run dev` to start development.
- `pnpm run build` to build the project.
- `pnpm run test` to run jest testing.
